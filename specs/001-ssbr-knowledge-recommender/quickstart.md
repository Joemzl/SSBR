# Quick Start Guide: SSBR官能化知识库与智能推荐系统

**Date**: 2026-02-28 | **Branch**: `001-ssbr-knowledge-recommender`

## 🎯 系统概述

本系统是一个基于 AI 对话助手的 SSBR 官能化知识库与智能推荐系统，包含：
- **4 套表征解读技能**：自动解读核磁、TEM、应力-应变、DSC 图片
- **1 套智能推荐技能**：根据性能需求推荐官能化方案

## 🚀 快速开始

### 1. 准备工作

#### 1.1 准备 PDF 文献

将 PDF 文献放入 `/literature/` 文件夹，命名规范：

```
{DOI}_{第一作者}_{年份}.pdf
```

示例：
- DOI: `10.1039/c9ra02783a`
- 命名: `10.1039_c9ra02783a_Gao_2019.pdf`

> ⚠️ DOI 中的 `/` 需替换为 `_`

#### 1.2 准备 Excel 数据

在 `/dataset/数据.xlsx` 中录入基础信息：

| 必填字段 | 说明 |
|----------|------|
| 样本ID | 唯一标识，如 `SSBR-001` |
| 样品名称 | 文献中的样品编号 |
| 官能化试剂名称 | 使用的试剂全称 |
| 核心官能团 | 关键功能基团 |
| 文献DOI | 对应 PDF 的 DOI |
| 图注信息 | 各表征图片的图注（如 `Figure 3a`） |

---

### 2. 使用表征解读技能

#### 2.1 解读 DSC 谱图

在 AI 对话中输入：
```
请解读第 3 行样本的 DSC 谱图
```

或：
```
请解读样本 SSBR-003 的 DSC 谱图
```

系统将自动：
1. 读取 Excel 中该行的 DOI 和图注
2. 在 `/literature/` 中匹配对应 PDF
3. 定位 DSC 谱图并解读
4. 输出标准化结果（可直接复制到 Excel）

#### 2.2 其他表征类型

| 表征类型 | 示例命令 |
|----------|----------|
| ¹H 核磁 | `请解读第 3 行样本的核磁谱图` |
| TEM | `请解读第 3 行样本的 TEM 图片` |
| 应力-应变 | `请解读第 3 行样本的应力-应变曲线` |
| DSC | `请解读第 3 行样本的 DSC 谱图` |

---

### 3. 使用智能推荐技能

#### 3.1 基本查询

```
我想用 SSBR 做轮胎胎面，希望 100%定伸应力达到 4.0MPa，请给我推荐官能化方案
```

#### 3.2 多指标查询

```
我想要 100%定伸应力在 3.5-4.5MPa 之间、Tg 达到 -30℃ 的官能化方案
```

#### 3.3 设置偏差容忍

```
我想要 100%定伸应力 = 4.0MPa 的方案，允许 ±10% 偏差
```

#### 3.4 浏览知识库数据

```
查看知识库全部数据
```

```
查看所有用于轮胎胎面的样本
```

```
查看样本 SSBR-001 的详细信息
```

## ⚠️ 注意事项

### 数据质量红线

1. **零编造**：所有数据必须源自文献
2. **零估算**：未标注数据用区间表示，不给单值
3. **零幻觉**：无文献支撑不输出

### 常见错误处理

| 错误提示 | 解决方案 |
|----------|----------|
| `未找到DOI为XXX的对应文献` | 检查 PDF 命名是否符合规范 |
| `未找到图注为XXX的图片` | 检查 Excel 中图注信息是否正确 |
| `知识库暂无数据` | 先通过解读技能录入数据 |

## 🔧 数据录入完整指南

### Step 1: 准备 PDF 文献

1. 获取目标文献的 PDF 文件
2. 确认文献包含 SSBR 官能化相关内容
3. 按命名规范重命名文件：
   ```
   {DOI}_{第一作者姓氏}_{年份}.pdf
   ```
4. 将文件复制到 `/literature/` 文件夹

**命名示例**：
| DOI | 第一作者 | 年份 | 文件名 |
|-----|----------|------|--------|
| 10.1039/c9ra02783a | Gao | 2019 | `10.1039_c9ra02783a_Gao_2019.pdf` |
| 10.1016/j.polymer.2020.122456 | Zhang | 2020 | `10.1016_j.polymer.2020.122456_Zhang_2020.pdf` |

### Step 2: 录入基础信息到 Excel

打开 `/dataset/数据.xlsx`，新增一行并填写以下字段：

#### 2.1 必填字段

| 字段 | 格式 | 示例 |
|------|------|------|
| **样本ID** | `SSBR-XXX` | `SSBR-005` |
| **样品名称** | 文献中的名称 | `SSBR-g-APTES` |
| **官能化试剂名称** | 试剂全称 | `3-氨丙基三乙氧基硅烷` |
| **核心官能团** | 官能团简写 | `-NH₂` |
| **文献DOI** | 完整 DOI | `10.1039/c9ra02783a` |

#### 2.2 图注信息字段

| 字段 | 格式 | 示例 |
|------|------|------|
| 核磁谱图 | 文献图注 | `Figure 2a` |
| 微相分离图片表征 | 文献图注 | `Figure 5c` |
| 应力-应变曲线 | 文献图注 | `Figure 8a` |
| DSC谱图 | 文献图注 | `Figure 3` |

#### 2.3 可选字段

| 字段 | 格式 | 示例 |
|------|------|------|
| 应用场景 | 预设值 | `轮胎胎面` / `密封制品` / `减震材料` / `通用橡胶制品` |
| 官能化程度 | 百分比数值 | `3.6` |
| 第一作者 | 姓氏 | `Gao` |
| 发表年份 | 年份 | `2019` |

### Step 3: 使用解读 Skill 获取标准化数据

依次调用 4 套解读 Skill，获取标准化解读结果：

```
请解读第 5 行样本的核磁谱图
请解读第 5 行样本的 TEM 图片
请解读第 5 行样本的应力-应变曲线
请解读第 5 行样本的 DSC 谱图
```

### Step 4: 将解读结果录入 Excel

从每个解读结果中提取核心数据，填入 Excel 对应列：

#### 力学性能数据

| 字段 | 单位 | 录入格式 |
|------|------|----------|
| 100%定伸应力 | MPa | 精确值 `4.2` 或区间 `4.0-4.5` |
| 200%定伸应力 | MPa | 精确值或区间 |
| 300%定伸应力 | MPa | 精确值或区间，无则填 `-` |
| 拉伸强度 | MPa | 精确值或区间 |
| 断裂伸长率 | % | 精确值或区间 |
| **力学数据来源** | - | `Table 1` / `图面标注` / `曲线估读` |

#### 热学性能数据

| 字段 | 单位 | 录入格式 |
|------|------|----------|
| 玻璃化转变温度Tg | ℃ | 精确值 `-45.2` 或区间 `-48至-42` |
| **热学数据来源** | - | `Table 2` / `图面标注` / `曲线估读` |

#### 表征解读结果

| 字段 | 内容 |
|------|------|
| 核磁解读 | 复制 NMR Skill 输出的完整解读 |
| TEM解读 | 复制 TEM Skill 输出的完整解读 |
| 应力应变解读 | 复制应力-应变 Skill 输出的完整解读 |
| DSC解读 | 复制 DSC Skill 输出的完整解读 |

### Step 5: 验证新增数据

录入完成后，执行以下验证：

1. **唯一性检查**：确认样本ID 不重复
2. **推荐测试**：使用智能推荐查询新增样本
   ```
   查看样本 SSBR-005 的详细信息
   ```
3. **匹配测试**：验证新数据可被推荐
   ```
   我想要 Tg 约 -45℃ 的官能化方案
   ```

## 📋 数据录入检查清单

每次录入新样本后，确认以下项目：

### 基础检查

- [ ] 样本ID 唯一（格式：`SSBR-XXX`）
- [ ] PDF 文件已放入 `/literature/` 文件夹
- [ ] PDF 命名符合规范（`DOI_作者_年份.pdf`）
- [ ] 文献DOI 与 PDF 文件名一致

### 字段检查

- [ ] 必填字段完整（样本ID、样品名称、试剂名称、核心官能团、DOI）
- [ ] 图注信息与文献实际图注一致
- [ ] 应用场景为预设值之一

### 数据质量检查

- [ ] 力学数据有来源标注（Table/图面标注/曲线估读）
- [ ] 热学数据有来源标注
- [ ] 无数据的字段填写 `-`
- [ ] 估读数据使用区间表示，非单值

### 解读检查

- [ ] 至少完成 1 套表征解读
- [ ] 解读结果已复制到 Excel 对应列

## 📈 数据来源层级说明

录入数据时，请标注数据来源以便推荐系统加权排序：

| 层级 | 标识 | 来源 | 可信度 | 示例标注 |
|------|------|------|--------|----------|
| **L1** | 最高 | 文献表格明确给出 | ★★★★★ | `Table 1` |
| **L2** | 高 | 图面直接标注数字 | ★★★★☆ | `图面标注` |
| **L3** | 中 | 曲线坐标估读 | ★★★☆☆ | `曲线估读 (约X-Y)` |
| **L4** | 低 | 文献未提供 | 不参与匹配 | `-` |

**重要**：L3 层级数据**必须使用区间表示**，禁止给出单值精确数字。

## 🔄 批量录入工作流

当需要从单篇文献录入多个样本时，推荐以下工作流：

```
1. 准备 PDF 文件
   └── 确认命名规范 → 复制到 /literature/

2. 批量录入基础信息
   └── 在 Excel 中连续添加多行
   └── 每行一个样本，填写基础字段

3. 批量执行解读
   └── 依次调用解读 Skill
   └── 建议按行号顺序执行

4. 批量复制解读结果
   └── 将各 Skill 输出复制到 Excel

5. 统一验证
   └── 查看知识库全部数据
   └── 检查新增样本是否正确显示
```
